Secure your GitHub Actions with Pinner

Automatically pin mutable tags to immutable SHA-1 hashes to prevent supply chain attacks.

Get Started View on GitHub
bash — pinner

$ pinner pin

Searching for workflows in .github/workflows/...

actions/checkout@v3 -> actions/checkout@df4cb1c0... # v3

dtolnay/rust-toolchain@master -> dtolnay/rust-toolchain@e97e2d8c... # master

Successfully pinned 12 actions across 3 files!

High Performance

Built with Rust for maximum speed. Scan and pin hundreds of actions in milliseconds.

Secure by Design

Protects your CI/CD against tag-moving attacks by switching to immutable SHAs.

Surgical Updates

Uses Regex-based replacement to preserve your YAML comments, indentation, and formatting.

Install Pinner

cargo install pinner
